# RustingRod — Mi camino hacia Rust

Repositorio de seguimiento del aprendizaje de Rust de Rodhan.

## Objetivo

Dominar Rust desde una base de C/Python para aplicarlo en proyectos reales.

## Estructura del repo

- `ROADMAP.md` — Plan de aprendizaje progresivo en 6 fases
- `PROGRESS.md` — Seguimiento de avance semanal y métricas
- `SESSIONS.md` — Registro de sesiones de estudio
- Ejercicios locales en `/home/rodri/rustlings/` (no versionados)

## Cómo usar este repo

Este repositorio documenta mi proceso de aprendizaje de Rust. Los ejercicios se resuelven localmente usando rustlings y 100 Rust Exercises. Aquí registro:

- Progreso semanal
- Conceptos aprendidos
- Errores frecuentes y soluciones
- Notas de cada sesión de estudio

El código de ejercicios no se versiona — solo la documentación del aprendizaje.
