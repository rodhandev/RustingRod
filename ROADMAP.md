# Roadmap de Aprendizaje Rust

Plan de aprendizaje progresivo estructurado en 6 fases.

## Fases

| Fase | Temas | Estado |
|------|-------|--------|
| 1 - Fundamentos | Ownership, borrowing, lifetimes, tipos básicos, pattern matching | 🔄 En curso |
| 2 - Estructura | Structs, enums, traits, generics, iteradores | ⏳ Pendiente |
| 3 - Sistemas | Result/Option, I/O, file system, procesos | ⏳ Pendiente |
| 4 - Concurrencia | Threads, Arc/Mutex, channels (mpsc), Send/Sync | ⏳ Pendiente |
| 5 - Async | tokio, async/await, futures, runtimes | ⏳ Pendiente |
| 6 - Aplicado | CLI tools (clap), servicios web (axum/actix), FFI con C | ⏳ Pendiente |

## Recursos usados

- **rustlings** — Ejercicios interactivos para aprender Rust de forma práctica
- **100 Rust Exercises** — Ejercicios estructurados para consolidar conceptos

## Metodología

Cada fase incluye:
1. Lectura de conceptos en The Rust Book
2. Ejercicios prácticos en rustlings
3. Ejercicios adicionales de 100 Rust Exercises
4. Proyecto mini aplicando los conceptos
